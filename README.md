# Password Validator - Quick Start

**Purpose:** Validate passwords based on rules in a file.

## Key Components

- **FileManager:** 
  - Validates and reads file content.

- **PasswordValidator:** 
  - Checks passwords against specified rules.
  - Counts valid passwords.

## Usage

1. **Run Validator:** 
   ```bash
   python main.py
   
2. **Run Tests:** 
   ```bash
   python -m unittest
